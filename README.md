# VSUL CO2 Solubility and Density Model

This repository contains the Python implementation used in:

Warisa N. (2026)
Benchtop Visualization of Carbon Storage Processes

The model calculates:

- CO2 solubility in water and NaCl brines
- CO2-saturated brine density
- Density increase due to dissolved CO2

using a modified Søreide–Whitson equation of state.
